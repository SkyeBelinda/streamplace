If you're seeing this page, you've manually navigated to a domain we use for local Streamplace development.

If you expected to see your local Streamplace installation here, that probably means you need to add an entry to your /etc/hosts file.

That entry would look something like...

192.168.99.99

...but it depends on the IP address of your local minikube cluster. Maybe you should ask around on Slack?
